# Shopping-Cart-WebPage

A simple and modern fashion store website built using HTML and CSS.  
It has a clean white background with black text and red accents to make it more attractive and easy to read.  
The website includes a homepage, shop, contact, and about sections.

## Features
- Clean and modern design  
- Fully responsive layout  
- Product cards with hover effects  
- Simple navigation bar  
- Banner and newsletter section  

## Technologies Used
- HTML  
- CSS
